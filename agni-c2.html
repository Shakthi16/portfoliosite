AGNI C2

Cybersecurity | AI Platform

2026

AGNI C2 is a next-generation Command & Control framework that integrates Local Large Language Models directly into the offensive security workflow. The platform was designed to reduce the cognitive overload of managing multiple beacons while providing real-time, explainable AI assistance for red team operators.

Domains:

Offensive Security AI Integration Full-Stack Dev Data Visualization

Tech Stack:

React / TypeScript FastAPI Sliver C2 Ollama (Qwen)

Challenge:

Modern red team operations suffer from tool fragmentation and cognitive overload. Operators lack a unified platform that correlates live campaign data with the MITRE ATT&CK framework while providing real-time tactical intelligence.

Role:

Sole researcher and full-stack developer. Responsible for the complete system architecture, from the React/Tailwind frontend to the FastAPI/WebSocket backend and the seamless integration of Sliver and Local LLMs.

The Vision Shift

The core problem with existing frameworks wasn't execution—it was cognition. Traditional setups map: Operator → Commands → Target. AGNI introduces a cognitive layer: Operator → AI Intelligence → Risk Analysis → Visualization → Target.

Core Features

1. The AI Mentor

Powered by local LLMs (Qwen) via Ollama, the AI Mentor provides real-time SSE streaming assistance. It understands the operational context and suggests OPSEC-safe commands while automatically mapping actions to the MITRE ATT&CK framework.

[ Fullscreen Screenshot: AgniAI Interface ]

2. Interactive Attack Graph Engine

An SVG-based visualizer for attack paths. Nodes represent compromised systems while edges represent MITRE techniques. The system calculates risk and allows the AI to suggest optimal lateral movement paths directly from the graph.

[ Fullscreen Screenshot: Attack Graph Visualizer ]

3. Defensive Bridge

Bridging the gap between Red and Blue teams. AGNI can automatically generate Sigma, YARA, and Suricata detection rules based on the attack techniques executed during a campaign.

[ Fullscreen Screenshot: RedBlueBridge Interface ]
